Automated Rover

391AH Honors Discovery Seminar

A color detecting and course adjusting rover. Built using DC motors, an Arduino-Uno processor, and a Pixy2 camera. The code that detects the position and color of the card is written in C++ and allows the rover to make automatous choices. This project involved research, design, developement, and an oral presentation that included a live demo.

Reverse Engineered Quadcopter

Side Project with Friends

I was part of a team that reverse engineered an RC drone by analyzing its UDP and TCP packets with Wireshark. We were able to reconstruct the commands in Python and transmit them ourselves using socket programming and threading. I researched the TCP packets responsible for video transmission, and also helped implement the configuration for a PS5 controller.
